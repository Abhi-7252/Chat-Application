# 💬 Chat Application (WebSocket)

## ▶️ Run on Localhost

Follow these steps to run the project locally:

1. Install dependencies:
npm install

2. Start the server:
npm run dev

3. Open your browser and go to:
http://localhost:4000
A real-time chat application built using Node.js, Express, and Socket.IO.

## 🚀 Features
- Real-time messaging
- Typing indicator
- Online users count
- Auto scroll to latest message

## 🛠️ Tech Stack
- Node.js
- Express.js
- Socket.IO
- HTML, CSS, JavaScript
